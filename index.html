🔒 Access control

Admin — logs in via the Admin tab with Firebase email + password. Change credentials at console.firebase.google.com → Authentication → Users.

Clients — log in via the Client tab with the password set here. No email or Firebase account needed.

⚠️ Required: Firestore security rules

Clients do not use Firebase Auth. For them to see products, your Firestore rules must allow unauthenticated reads. Go to console.firebase.google.com → Firestore → Rules and set:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Products & config: anyone can read, only auth users can write
    match /products/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    match /app_config/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
